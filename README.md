# Student Tracking System

A full-stack web application to manage student records with authentication functionality.  
Built as a mini project using Node.js and JavaScript.


## Features

- 🔐 User Authentication (Login system)
- ➕ Add new student records
- ✏️ Update existing student details
- ❌ Delete student records
- 📋 View all students
- 🗄️ Data stored using SQL database

---

## Tech Stack

- Backend: Node.js, Express.js
- Frontend: HTML, CSS, JavaScript
- Database: SQL

## How to Run
1. Clone the repository:
git clone https://github.com/anaghachebbi/student-performance-and-attendance-tracker

2. Install dependencies:
npm install

3. Start the server:
node server.js

4. Open in browser:
http://localhost:3000
